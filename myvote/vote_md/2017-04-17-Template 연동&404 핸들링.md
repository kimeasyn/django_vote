#### Detail/Template 연동&404 핸들링



우선 간단하게 detail.html 템플릿을 생성한다

```html
<--myvote/vote/templates/vote/detail.html-->
{{ question }}
```



view.py에 detail 함수 수정

```python
#views.py
def detail(request, question_id):
    question = Question.objects.get(pk = question_id)
    return render(request, 'vote/detail.html', {'question': question})
	#'question' 이란 이름으로 question 객체를 request로 던져준다.
    #'question' 이란 이름의 question 객체는 detail.html에서 question 이란 이름의 객체로 전달 받는다
```



404page 핸들링

```python
# views.py
from django.http import HttpResponse, Http404

def detail(request, question_id):
    try:
        question = Question.objects.get(pk = question_id)
    except Question.DoesNotExist:
        raise Http404("Qeustion %s does not exist" % question_id)
        #(id에 해당하는)Question 객체가 없을 경우404page와 함꼐 보여줄 메세지
    return render(request, 'vote/detail.html', {'question': question})
```
